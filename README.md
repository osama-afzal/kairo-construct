# The Kairo Construct

An agent-based computational sandbox modeling structural alienation and the fragile dynamics of human attachment under conditions of digital insulation. Inspired by Kiyoshi Kurosawa’s cinematic masterpiece *Kairo* (2001), the simulation reduces the human social fabric to a field of identical, autonomous white nodes traversing a closed, entropic void. 

For a comprehensive deep-dive into the sociological, psychological, and programmatic foundations of this ecosystem, see the official **[Kairo Construct Manifesto](./kairo-construct-manifesto.md)**.

## Overview

The **Kairo Construct** is a high-performance, canvas-based behavioral simulation where autonomous entities move in a continuous 2D space. Moving far beyond basic physical particle systems, the project layers deterministic psychological metrics and persistent memory states directly over the coordinates to explore emotional boundaries, codependency, and systemic burnout.

## System Behavior

The simulation operates on a strict frame clock sequence, completely decoupling high-frequency multi-agent data states from React's rendering lifecycle to eliminate re-render overhead:

1. **Sensory Pass**: Agents evaluate their local radar horizons ($120\text{px}$) to track physical proximity and compute the absolute nearest living node.
2. **Persistent Relationship Layer**: Decouples social states from loose spatial proximity using an order-independent database map with strict mutual exclusivity constraints. Two agents will completely reject bonding if either is committed to a partner elsewhere.
3. **Critical Drive Breakaways**: Attached agents execute proactive database overrides, forcefully deleting their relationship rows to balance personal autonomy or escape absolute isolation panic based on hardwired drive thresholds.
4. **Relational Metrics Tax**: Live database rows override raw distance calculations, steadily consuming an agent's *Social Battery* while successfully anchoring and flushing out their *Loneliness Index*.
5. **Exclusivity Vitality Lifeline**: Systemic life force (*Vitality*) is completely disconnected from internal stress meters. Agents are entirely blind to their own mortality; health regenerates passively inside database relationships ($+0.15/\text{frame}$) and undergoes a continuous void bleed when unattached ($-0.03/\text{frame}$).
6. **Secure-Secure Resonance Matrix**: Features tax-immune secure anchors that generate infinite energetic loops, locking their vectors directly into fluid, synchronized trajectories ($45\text{px}$ scale) to act as stable binary systems that sustain the ecosystem.
7. **The Blasé Attitude Mutation**: If an agent fails to manage its metrics and its battery drops to $3\%$, it undergoes a systemic collapse. It turns off its steering, purges all active relationship memory, and drops out of the lifeline—bleeding vitality down to $0\%$ as an inert grey monument unless rescued.

## Running the Project

Ensure you have [Node.js](https://nodejs.org) installed (v18 or higher recommended).

### 1. Clone and Navigate
```bash
git clone https://github.com
cd kairo_construct
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```
Open your browser and navigate to the local server URL (usually `http://localhost:5173`) to observe the simulation.

## Concept

This project acts as an interactive, mechanistic essay on human geometry and existential double-binds, validating that long-term systemic equilibrium and survival can emerge entirely as an un-coerced byproduct of agents navigating the baseline human tension between the hunger for personal autonomy and the terror of absolute abandonment.
